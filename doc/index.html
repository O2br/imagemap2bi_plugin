My first Python Qgis Plugin

Introduction

With this plugin you can write an Image Map from current map in the format compatible with MicroStrategy

Versions

  • 0.0.1 (02-2018): initial first beta version derivative forked the famous 'Html Image Map Plugin' from Richard Duivenvoorde

Download

You will need Qgis > version 0.9, and the python bindings installed. Download: see above for different versions.
